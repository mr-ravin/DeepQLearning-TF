# DeepQNetwork-TF
This repository includes source code for implementing Deep Reinforcement Learning architecture (DQN) using Tensorflow library.

#### Code Developer: [Ravin Kumar](https://mr-ravin.github.io)
#### Requirements:
- #### Tensorflow: 1.12
- #### Python: 3    
- #### gym environment

### Training Steps:
```python3
python3 dqn.py
```
